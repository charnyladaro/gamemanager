File Storage Information

Upload Directories:

  • Game Files: backend/uploads/games/
  • Cover Images: backend/uploads/covers/

File Management Tips:

  • Files are automatically renamed with timestamps when uploaded
  • Deleting a game will NOT automatically delete its files from disk
  • Use the cleanup tool regularly to remove orphaned files
  • Maximum upload size: 500GB per file
  • Supported game formats: .exe, .msi, .zip, .rar, .7z
  • Supported image formats: .png, .jpg, .jpeg, .gif, .webp

⚠️ Important Recommendations

  • Regular Backups: Create regular backups of your uploads folder and database
  • Disk Space: Monitor available disk space to prevent upload failures
  • File Verification: Periodically verify that game files are not corrupted
  • Security: Ensure proper file permissions on upload directories
